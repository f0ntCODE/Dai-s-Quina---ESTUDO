# Dai-s-Quina---ESTUDO
Um sistema de e-commerce para uma varejista de móveis, eletrônicos e eletrodomésticos. Esse projeto tem como objetivo treinar o desenvolvimento full-stack passando por todas as etapas de confecção, como levantamento de requisitos, análise de requisitos, engenharia de software, design, lógica de programação, banco de dados, testes e deploy
