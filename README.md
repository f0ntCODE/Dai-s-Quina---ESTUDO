# Projeto Dai's Quina: "Transformando o seu lar"

## O que este projeto propõe
Este projeto tem como objetivo praticar o desenvolvimento de software web full-stack, colocando em prática as *skills* já desenvolvidas ao longo dos estudos e aprendendo novas à medida em que se cria o projeto.

## Como está estruturado esse desafio
Desenvolver software está além de "fazer telinhas" ou "ficar no ar condicionado fazendo programa". É um processo complexo, trabalhoso e exige muito do intelecto e da capacidade de compreender o problema a ser resolvido, especialmente para desenvolvedores full-stack. 
Estas etapas fazem parte da jornada desse desafio:

- Levantamento de requisitos;
- Análise de requisitos;
- Engenharia de software;
- Design gráfico;
- Codificação;
- Testes;
- *Deploy*;

Todas essas etapas serão colocadas à prova e serão melhoradas com o tempo.

> OBS: Esse projeto não contempla nenhum e-commerce verdadeiro, sendo usado para aprendizagem e portifólio.
